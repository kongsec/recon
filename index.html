❓ How to Use

Step-by-Step

  1. Enter target domain (e.g., gov.uk)
  2. Click file patterns to select (env.js, config.js, etc.)
  3. Click "Generate Search Links"
  4. Click "Open" links to search URLScan.io
  5. On URLScan: Click a result → HTTP tab → "Show response"
  6. Copy the response content
  7. Paste into Content Analyzer → Click "Analyze"
  8. Copy/Download your report!
